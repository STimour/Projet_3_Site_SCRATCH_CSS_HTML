<!DOCTYPE html>

Ce n'est pas une balise. Cependant l'inclusion du DOCTYPE dans un document HTML assure que le navigateur interprétera la version HTML. Il s'agit d'une "information" pour le navigateur sur le type de document à attendre.

Attribus

Nom Description Valeurs (Exemples)
html Est une déclaration du format <!DOCTYPE html> 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, 
    initial-scale=1.0">
    <title>Document</title>
</head>
<body>  


</body>
</html>
